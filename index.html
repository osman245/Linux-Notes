Unix Commands & Utilities

A practical LFCS-style reference covering the Unix commands and utilities you will encounter most frequently as a Linux system administrator.

Log in to Local & Remote Consoles

Secure remote access is performed using SSH, which encrypts credentials and protects against eavesdropping. 

ssh user@host
ssh -v user@host
ssh -V

Read and Use System Documentation

man ls
man -k network
apropos disk
sudo mandb

Create, Delete, Copy, and Move Files

ls -lah
pwd
mkdir lfcs
touch lfcs.txt
cp -ar /tmp/invoice /home/bob
rm -rf /home/bob/lfcs

Create and Manage Links

Hard links share the same inode. Soft links act as aliases and can link directories. 
ln file1 file2
ln -s /tmp /home/bob/link_to_tmp

List, Set, and Change Permissions

chmod 644 file
chmod go+r file
chown -R alice:developers /path

SUID, SGID, and Sticky Bit

chmod u+s,g+s,o+t /home/bob/datadir

Search for Files

find . -name file.txt
find -mmin -5
find -perm 664
sudo find /usr -size +5M -size -10M

Compare and Manipulate File Content

head -5 file
tail file
sed 's/canda/canada/g' userinfo.txt
cut -d',' -f3 userinfo.txt
uniq countries.txt

Search Files with Grep & Regex

grep 'password' /etc/ssh/sshd_config
grep -ri 'password' /etc
grep '^#' /etc/login.defs
egrep '/dev/.*' /etc

Archive, Backup, and Compress

tar cf archive.tar file
tar xf archive.tar
tar czf archive.tar.gz file
zip -r archive.zip dir/

Sync Directories & Disk Imaging

rsync -a dir/ user@host:/path/
sudo dd if=/dev/vda of=diskimage.raw bs=1M status=progress

Input & Output Redirection

command > output.txt
command 2> error.txt
command > all.txt 2>&1

Working with SSL / TLS

openssl req -newkey rsa:2048 -keyout key.pem -out req.pem
openssl req -x509 -newkey rsa:4096 -days 365

Git Basic Operations

git init
git add .
git commit -m "Initial commit"
git branch feature
git checkout feature
git merge feature
git log
