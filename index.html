02

Operations & Deployment

Console access, file management, text processing, archiving, networking security, and version control fundamentals.

Console Access & SSH

Access locally or remotely. Remote access must be secured via SSH to encrypt credentials in transit.

  • ssh -V
  • ssh username@node
  • sudo -v ssh@alex

System Documentation

Use --help and man. Search manuals with man -k. Update index with mandb.

  • man -k keyword
  • mandb
  • sudo mandb
  • sudo hostnamectl set-hostname web-server-01

File & Directory Management

Create, copy, move and remove files and directories using core commands.

  • ls -lah
  • cp -ar /tmp/invoice /home/bob
  • rm -rf /home/bob/lfcs
  • mkdir -p /1/2/3/4
  • ls --full-time

Links (Hard & Soft)

Hard links point to the same inode. Soft links act as aliases and can cross filesystems.

  • ln -s /tmp /home/bob/link_to_tmp
  • ln /tmp /home/bob/link_to_tmp

Permissions & Special Bits

Modify permissions using numeric or symbolic modes. Special permissions: SUID, SGID, Sticky Bit.

  • chmod 644 filename
  • chown -R alice:developers /path/to/directory
  • chmod go+r filename
  • chmod u+s,g+s,o+t /home/bob/datadir

Finding Files

  • find -name file1.txt
  • find -mmin -5
  • find -name "f*" -size +512k
  • sudo find /usr -type f -size +5M -size -10M > /home/bob/size.txt
  • find . -perm 664

Text Processing

  • head -5 myfile
  • tail -n 10 /var/log/apt/term.log
  • sed 's/canda/canada/g' userinfo.txt
  • cut -d ' ' -f 1 userinfo.txt
  • uniq countries.txt
  • diff -c file1 file2

Grep & Regex

  • grep -rwi 'password' /etc/ssh/sshd_config
  • grep -w '7' /etc/login.defs
  • egrep -r '0{3,}' /etc/

Archiving & Compression

  • tar -tf archive.tar
  • tar czf archive.tar.gz file1
  • zip -r archive.zip Pictures/
  • gzip file1
  • bzip2 file2
  • xz file3

Rsync & Remote Sync

  • rsync -a Pictures/ aaron@9.9.9.9:/home/aaron/Pictures/
  • rsync -a aaron@9.9.9.9:/home/aaron/Pictures/ .

Redirection & Streams

  • date > file.txt
  • grep -r '^The' /etc/ 2>/dev/null
  • stdin (<), stdout (1>), stderr (2>)

SSL / TLS with OpenSSL

  • man openssl
  • openssl req -newkey rsa:2048 -keyout key.pem -out req.pem
  • openssl req -x509 -noenc -newkey rsa:4096 -days 365

Version Control (Git)

  • sudo apt install git
  • git init
  • git add .
  • git commit -m "changes"
  • git branch 1.1-testing
  • git switch 1.1-testing
  • git merge 1.1-testing
  • git remote -v
  • ssh-keygen
  • git log
